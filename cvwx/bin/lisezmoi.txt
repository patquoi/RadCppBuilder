2. DÉSINSTALLATION D'UNE VERSION PRÉCÉDENTE
--------------------------------------------------------------------------------

SI VOUS AVEZ INSTALLÉ UNE VERSION PRÉCÉDENTE DE CENTRE-VILLE, IL EST FORTEMENT
CONSEILLÉ DE LA DÉSINSTALLER AVANT D'INSTALLER CETTE VERSION.

Attention : les fichiers CRÉÉS NE SERONT PAS SUPPRIMÉS. Seuls les fichiers 
installés (même modifiés) sont supprimés. Donc, si vous avez modifié les
exemples fournis Géant.cvw, Carfours.cvw ou Démo.cvw, il faut les sauvegarder
avant ou les renommer (au menu principal, rubrique Fichier | Enregistrer sous).

------------------------------------------------------------------------------
3. SI VOUS AVEZ DES FICHIERS CRÉÉS AVEC UNE VERSION ANTÉRIEURE DE CENTRE-VILLE
------------------------------------------------------------------------------

 - À PARTIR DE LA VERSION 3.0, IL N'EST PLUS POSSIBLE D'OUVRIR DES FICHIERS 
   DE SIMULATION CRÉÉS AVEC UNE VERSION ANTÉRIEURE À LA VERSION 2.0 (EXCLUE).
 - DANS LES AUTRES CAS, LA MISE À NIVEAU DES FICHIERS SE FAIT AUTOMATIQUEMENT
   AU CHARGEMENT AVEC TOUTE NOUVELLE VERSION.

6. CONTENU DES SOURCES DE PROGRAMME
------------------------------------------------------------------------------

Centre-Ville est en GPL depuis la version 3.7.
Ceci entraîne le fait que les archives contiennent les sources des programmes.
Les sources sont en C++Builder 6. Les sources de la version 3.1 ont été 
incluses en guise d'exemple de portage Delphi. Les sources de la version 4.3.1
ont été également incluses en guise d'exemple de code C++ commun entre les 
versions Windows et Linux. Le projet Kylix Delphi (v3.1) peut être ouvert avec
toutes les versions de Kylix alors que le projet Kylix C++ (v4.3.1) ne peut 
être ouvert qu'avec Kylix 3 ou version supérieure. Pour plus d'informations, 
consulter le fichier INFOSRC.TXT.
