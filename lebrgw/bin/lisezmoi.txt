INFORMATIONS IMPORTANTES - LE BERGER
------------------------------------

PLATEFORMES

De Windows XP à Windows 10 et plus.

REGLE ET UTILISATION

Pour avoir de l'aide et connaître la règle du jeu, il suffit
d'appuyer sur la touche F1 ou d'aller au menu Informations | 
Règle du jeu et utilisation. Il est fortement recommandé de 
consulter d'abord l'aide en ligne avant de commencer à 
utiliser Le Berger.


HISTORIQUE DES VERSIONS (CORRECTIONS ET AMELIORATIONS)

Les versions inférieures à la version 1.0 n'incluent pas la 
notion de partie (mode jeu) : il n'y a que le mode éditeur. 
Cependant, il est possible de tester n'importe quel niveau 
dans le mode éditeur.

[1.7] Versions 32 et 64 bits
 - Les fichiers de configuration et de niveaux créés se 
   trouvent à présent dans le dossier : 
	%LocalAppData%\Patquoi.fr\Le Berger
   et pour Windows XP : 
	%LocalAppData%\Local Settings\
		Application Data\Patquoi.fr\Le Berger
 - L'aide est à présent au format html et se trouve dans le
   dossier html où se trouve l'application.
 - Support de la haute résolution HDPI.
 - Le diplôme en fin de jeu peut être enregistré sous forme
   d'image de deux manières uniquement :
    - en exportant sous forme de fichier (Ctrl+E)
    - en copiant dans le presse-papier (Ctrl+C)
   au lieu d'être uniquement imprimable (Ctrl+I).
 - Changement de l'icône (haute résolution)

[1.4]
